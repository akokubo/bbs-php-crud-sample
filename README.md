# 掲示板サンプル

## システム
PHP 5.6以上

## 設定
1. データをWebサーバーにアップロード
2. dbフォルダをWebサーバーが書き込めるようにする
3. setup_sample_db.phpをコマンドラインから実行するか、Webからアクセスする
4. index.phpにWebからアクセスする

## 注意
libやtmplフォルダの.htaccessは、これらのフォルダがWebからアクセスできないように設定している
